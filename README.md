# **Peworld**

![](https://i.imgur.com/s9ywDPR.png)


**Equipments**
* Text Editor
* React JS
* Terminal
* Redux
* Tailwind
* Search Talent
* Recruit
* etc

**Prerequisites**
* Latest `NPM / YARN`
* Add enpoint REST API Client environment

**Install Peworld Web App**
```
https://github.com/acrossmindanduniverse/peworld.git

npm install / yarn install
```

**env**
```
REACT_APP_BASE_URL= (URL)
```
